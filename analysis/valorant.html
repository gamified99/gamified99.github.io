Valorant Game Modes: Analysis

UNRATED
Mode Type: Casual play, No rank impact, Standard 5v5
Core Mechanics: First to 13 rounds, Spike gameplay, Role switch at 12
Agent Setup: Unique agents, Pre-match selection

CUSTOM
Mode Type: Private match, Not ranked, No XP
Core Mechanics: Rule editing, Cheats toggle, Pause/overtime options
Practice Use: Scrims, Observer tools, Aim/strat training

DEATHMATCH
Mode Type: Free-for-all, No abilities, 14 players
Core Mechanics: Instant respawn, Silent movement, All weapons unlocked
Aim Practice: Headshot training, Spray control, Spawn prediction

TEAM DEATHMATCH
Mode Type: 4v4 teams, Abilities enabled, Custom maps
Core Mechanics: Instant respawn, Loadout phases, Map pickups
Practice Focus: Team coordination, Combat-ready agents, Fast aim drills

COMPETITIVE
Mode Type: Ranked 5v5, Unlocks at Level 20, Structured progression
Core Mechanics: MMR vs Rank, RR system, Overtime rules
Climb Factors: Performance-based RR, Party restrictions, Leaderboards

SWIFT PLAY
Mode Type: Short matches, First to 5 rounds, ~15 minutes
Core Benefits: Quick warm-up, Time-efficient, Low-stress
Player Gains: XP farming, Agent testing, Teamplay practice

PREMIER
Mode Type: Team-based league, Scheduled matches, Playoff system
Core Benefits: Esports-style format, Strategy building, Skill benchmarking
Pro Potential: Path to pro, High-level practice, Advanced coordination

ESCALATION
Mode Type: Team-based arcade, Gun progression, Fast-paced chaos
Core Mechanics: Strong-to-weak weapons, Randomized sets, Tier-based win
Player Gains: Aim versatility, Casual fun, High XP efficiency

SPIKE RUSH
Mode Type: Fast 5v5, First to 4 rounds, Random loadouts
Core Mechanics: All carry Spike, Power-up orbs, Full abilities
Player Gains: Quick warmups, XP farming, Casual fun
